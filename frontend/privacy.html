BriskMail Privacy Policy

English Português

Last updated: April 29, 2025

BriskMail does not collect, store, or sell any personal data. Email content is sent to our analysis API solely to generate a real-time response and is never retained.

1. Overview

BriskMail ("the Extension") is a Chrome browser extension that analyzes Gmail emails using artificial intelligence. This Privacy Policy explains what information the Extension accesses, how it is used, and what rights you have.

2. Data We Access

The Extension accesses the following information:

  • Email body text — read directly from the Gmail page you have open in your browser, only when you click the "Analyze Email" button.
  • Active tab URL — used only to detect whether the current tab is Gmail, in order to show the correct status in the popup.
  • Language preference — stored locally via chrome.storage.sync to remember whether you prefer Portuguese or English responses.

3. How We Use Your Data

  • Email content is transmitted over HTTPS to our analysis API (email-classifier-api.fly.dev) only when you explicitly click the button.
  • The API processes the text with an AI model and returns a summary, category, priority score, and reply suggestions.
  • No email content is stored, logged, or used for training. The API processes the request in memory and discards it immediately after returning the response.
  • The language preference is stored only on your device and is never sent to any server.

4. Data Sharing

We do not sell, rent, or share your personal data with third parties. The AI inference in production is powered by OpenAI, which processes the email text as part of the API request. OpenAI's data usage policy applies to that processing. We do not send any identifying information (name, email address, account ID) alongside the email text.

5. Data Storage and Retention

  • Email content is never stored on our servers or any external database.
  • Your language preference is stored locally in Chrome sync storage and can be changed or cleared at any time from the extension popup.
  • We do not use cookies, analytics trackers, or any persistent identifiers.

6. Permissions Justification

The Extension requests the following Chrome permissions:

  • activeTab — to read the email text from the Gmail tab you are viewing.
  • tabs — to detect whether Gmail is open, used only for the status indicator in the popup.
  • storage — to save your language preference locally.
  • host permission for mail.google.com — required to inject the "Analyze Email" button into Gmail.
  • host permission for email-classifier-api.fly.dev — required to send the email text to our analysis API.

7. Children's Privacy

BriskMail is not directed at children under the age of 13 and does not knowingly collect information from children.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by the "Last updated" date at the top of this page. Continued use of the Extension after changes constitutes acceptance of the updated policy.

9. Contact

If you have questions or concerns about this policy, please contact us at contatobriskmail@gmail.com.

O BriskMail não coleta, armazena nem vende nenhum dado pessoal. O conteúdo do email é enviado à nossa API de análise apenas para gerar uma resposta em tempo real e nunca é retido.

1. Visão Geral

O BriskMail ("a Extensão") é uma extensão do Chrome que analisa emails do Gmail usando inteligência artificial. Esta Política de Privacidade explica quais informações a Extensão acessa, como elas são usadas e quais são seus direitos.

2. Dados que Acessamos

A Extensão acessa as seguintes informações:

  • Texto do corpo do email — lido diretamente da página do Gmail aberta no seu navegador, somente quando você clica no botão "Analisar Email".
  • URL da aba ativa — usada apenas para detectar se a aba atual é o Gmail, a fim de exibir o status correto no popup.
  • Preferência de idioma — armazenada localmente via chrome.storage.sync para lembrar se você prefere respostas em português ou inglês.

3. Como Usamos seus Dados

  • O conteúdo do email é transmitido via HTTPS para nossa API de análise (email-classifier-api.fly.dev) somente quando você clica explicitamente no botão.
  • A API processa o texto com um modelo de IA e retorna um resumo, categoria, pontuação de prioridade e sugestões de resposta.
  • Nenhum conteúdo de email é armazenado, registrado ou usado para treinamento. A API processa a requisição em memória e descarta o texto imediatamente após retornar a resposta.
  • A preferência de idioma é armazenada apenas no seu dispositivo e nunca é enviada a nenhum servidor.

4. Compartilhamento de Dados

Não vendemos, alugamos nem compartilhamos seus dados pessoais com terceiros. A inferência de IA em produção é realizada pela OpenAI, que processa o texto do email como parte da requisição à API. A política de uso de dados da OpenAI se aplica a esse processamento. Não enviamos nenhuma informação identificável (nome, endereço de email, ID de conta) junto com o texto do email.

5. Armazenamento e Retenção de Dados

  • O conteúdo do email nunca é armazenado em nossos servidores ou em qualquer banco de dados externo.
  • Sua preferência de idioma é armazenada localmente no Chrome sync storage e pode ser alterada ou removida a qualquer momento pelo popup da extensão.
  • Não utilizamos cookies, rastreadores de analytics ou identificadores persistentes.

6. Justificativa de Permissões

A Extensão solicita as seguintes permissões do Chrome:

  • activeTab — para ler o texto do email na aba do Gmail que você está visualizando.
  • tabs — para detectar se o Gmail está aberto, usado apenas para o indicador de status no popup.
  • storage — para salvar sua preferência de idioma localmente.
  • host permission para mail.google.com — necessária para injetar o botão "Analisar Email" no Gmail.
  • host permission para email-classifier-api.fly.dev — necessária para enviar o texto do email à nossa API de análise.

7. Privacidade de Crianças

O BriskMail não é direcionado a crianças menores de 13 anos e não coleta intencionalmente informações de crianças.

8. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. As alterações serão refletidas pela data de "Última atualização" no topo desta página. O uso continuado da Extensão após alterações constitui aceitação da política atualizada.

9. Contato

Se você tiver dúvidas ou preocupações sobre esta política, entre em contato pelo email contatobriskmail@gmail.com.
